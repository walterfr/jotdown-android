# Jotdown — Fichador Acadêmico

Fichador acadêmico para Android que funciona 100% offline.

## Funcionalidades
- Leitor de PDF integrado
- Captura de citações e destaques
- Anotações tipo post-it
- Ferramentas de desenho (caneta, lápis, marca-texto, borracha)
- Exportação ABNT em .MD, .TXT e .PDF
- Armazenamento local via IndexedDB
